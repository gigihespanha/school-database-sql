# School Database SQL Project

This project demonstrates the creation and management of a relational database using Microsoft SQL Server.

## Project Overview
The database models a simple school system with students, professors, and disciplines. It includes:

- Database creation
- Tables with primary and foreign keys
- Sample data insertion
- Queries using JOIN, WHERE, and ORDER BY
- View creation

## Files Description

- **school_database.sql**  
  Complete script containing the full database setup.

- **schema.sql**  
  Database schema: tables, primary keys, and foreign keys.

- **seed_data.sql**  
  Sample data inserted into the tables.

- **views.sql**  
  SQL view combining disciplines and professors using JOIN.

- **queries.sql**  
  Example queries to retrieve and filter data.

## How to Run
1. Open SQL Server Management Studio (SSMS)
2. Execute `school_database.sql`  
   or run the scripts in this order:
   - `schema.sql`
   - `seed_data.sql`
   - `views.sql`
   - `queries.sql`
